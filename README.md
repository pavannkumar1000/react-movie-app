# React Movie App

This is a **React movie browsing app** built using **React + Vite**.

## Features

- Browse popular movies using **TMDB API**
- Search for movies
- Component-based architecture
- Environment variables used for **secure API key**
- Fully responsive design
- Deployed on **Netlify** (CI/CD from GitHub)

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS 
- TMDB API
- Netlify for deployment
